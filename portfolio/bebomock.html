Features and UX Suggestions for Bebo Mobile

Objective 1: Format Appropriately for Mobile


Problem: Current UI is hard to navigate due to lack of emphasis on important aspects and small buttons/text boxes.
Method:

  1. Make Bebo header bigger/more prominent
  2. Have a dashboard screen which has all the links (Home, Profile, Friends, etc) in icon styled links.
  3. Make “Say Something” Bar bigger and more easily accessible
  4. For Mail, add a small notification image showing number of new messages.
  5. Photo should just be an upload from phone or taking a picture using phone.

Objective 2: Remove any clutter

Problem: There are redundant links and a features not highly used on mobile.
Method:

  1. Remove “Edit Profile” when in “Profile” view. (Keep that for desktop Bebo)
  2. Remove “My Friends” when in “Profile” view (redundant, already is a main menu item)
  3. Remove as much text as possible, like “Say Something” (put that into the text box instead)
  4. (In Android) Add Feedback, Help, and Search to Menu

Objective 3: Keep Bebo’s Culture, but stay simple

Problem: Customization settings are complicated and some phrasing is ambiguous.
Method:

  1. Settings should let people customize the icon colors (which I mocked in red). The First B of Bebo could also change. (Only give uses mono-colored options, less hassle compared to link/box colors, and looks slick)
  2. Instead of “Home” as a dashboard item, it can be called “Lifestream.” More clear/direct
